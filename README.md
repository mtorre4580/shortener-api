# Shortener-API

Shortener API built with Prisma.io and Express.js

<img src="" alt="shortener-api">

## Installation

- `npm i`

## Run API in development

- `npm run dev`

## Build for production

- `npm run build`
- `npm run start`

## Tests

- `npm run test`

## Stack

- Express.js
- Prisma.io
- TypeScript
